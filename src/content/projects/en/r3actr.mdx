# 🌐 R3ACTR – Rewamped

The official **R3CTR** website — rebuilt from the ground up using **TypeScript**, **Astro**, and **Tailwind CSS** for speed, scalability, and modern design.  
This version ("Rewamped") represents our startup’s new digital identity — faster, cleaner, multilingual, and built for growth.

---

## 🚀 Overview

**R3ACTR** is a next-generation software development startup specializing in AI, ML, and web technologies.  
Our revamped website serves as our digital home — introducing who we are, what we build, and how to reach us.

---

## ✨ Features

- ⚡ **Built with Astro + TypeScript** for high-performance static rendering  
- 🎨 **Tailwind CSS** for rapid, consistent styling  
- 🌍 **Internationalization (i18n)** — multi-language support using custom translation utilities  
- 📨 **Functional Forms** — Newsletter & Contact forms (via Formspree or backend API)  
- 💎 **Reusable UI Components** — responsive cards, hero sections, and animated squircle backgrounds  
- 🧭 **SEO Optimized** — metadata, structured data, and fast load times  
- 🔒 **Production Ready** — easy deploy on Vercel, Netlify, or custom server
